# DJ-AGENT

A browser-based DJ set: dual decks, mixer, step sequencer, sampler pads, keys/bass, voice recording, sequence presets (JSON), and master mix export.

## Quick start

```bash
npm install
npm run dev
```

Open the URL Vite prints, click **Enable audio**, then load tracks or sequence `.json` files on the decks.

## Scripts

| Command       | Description      |
| ------------- | ---------------- |
| `npm run dev` | Dev server       |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |

## Stack

React, TypeScript, Vite, [Tone.js](https://tonejs.github.io/).
